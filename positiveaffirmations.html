Positive Affirmation Generator

My Role: UI Designer, full-stack developer, researcher; Type of Project: Personal; Software: Figma, VSCode, Photoshop



Challenge: To create a website that would generate various positive affirmations for the user.
What I did: I used Figma to create mockups for the generator. Then I used a combination of HTML, CSS, and JavaScript to program the generator using an array of positive words. The generator would use "I am", "I believe", and "I will" as prompts for the user to choose from.

Inspiration

2020 was an extremely hard year for everyone. I was inspired by love languages and some cannot be met at this time in our lives. There are five love languages but I wanted to focus on Words of Affirmation.
