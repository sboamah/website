Positive Affirmation Generator

My Role: UI Designer, Front-end developer, researcher; Type of Project: Personal; Software: Figma, VSCode, Photoshop

Problem Statement: Generation Z is lacking a resource for daily affirmations that cater to any situation or emotion.

Challenge: To create a website that would generate various positive affirmations for the user.
What I did: I used Figma to create mockups for the generator. Then I used a combination of HTML, CSS, and JavaScript to program the generator using an array of positive words. The generator would use "I am", "I believe", and "I will" as prompts for the user to choose from.

Inspiration

2020 was an extremely hard year for everyone. I was inspired by love languages and how some cannot be met at this time in our lives. There are five love languages but I wanted to focus on Words of Affirmation.


First Iteration




After my first iteration, I revived the project in 2021 after continuing to see a demand for affirming words in the midst of COVID-19. I wanted to target Generation Z in my user feedback solicitation:

Key User Feedback: 12/22/2021

    User 1: 23 years old
    Functionality works
    Affirmations are great and resonate:
      I am enough, I believe I deserve love, I believe in celebrating myself, I will never quit, I will not let others define me.
    UI suggested changes:
      Generate button should have a fixed position.
      Letter spacing should be wider.
      Drop down for form choices (I am, I believe, I will).
      Responsiveness to different devices

    User 2: 21 years old
    Functionality works
    Affirmations resonate:
      I will allow imperfection, I will be there for myself, I believe I am here for a reason.
    UI suggested changes:
      Letter spacing should be wider.
      Use of colors in the background and/or UI elements.


    From this feedback, I was able to implement these changes, target mobile use of the system, and add engaging colors.
