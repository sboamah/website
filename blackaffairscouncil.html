Black Affairs Council Website

My Role: UI Designer, full-stack developer, researcher; Type of Project: Academic Software: Canva, VSCode



Challenge: To create a website that would showcase something of social significance. "Individual Software Engineering" course.
What I did: Along with my partner, our goal was to create a website that would introduce the user to the Black Affairs Council, an organization at Stevens Institute of Technology intended to represent the Black community and the Black-centered organizations on campus. The website would JavaScript, HTML, and CSS. I was in charge of designing and coding the front-end of the website.



TIMELINE AND MILESTONES

March 26th - April 5th 2020: Final Decision on what website would look like
April 4th - April 5th 2020: Responsibilities on HTML, CSS, and Javascript code
April 4th - April 12th 2020: Completion of the FIRST draft of code (HTML and CSS). Testing begins with the basics of the website (user interface, pros, and cons)
April 12th - April 19th 2020: Completion of the SECOND draft of code. Testing on the page to page functionality (HTML, CSS, and Javascript).
April 19th - April 26th 2020: Completion of the FINAL draft of code. Testing on the overall website with all testers.

Inspiration

For our design, the biggest inspirations were the website for Alpha Phi Alpha Fraternity Inc. and Howard University. The focus was on wide images that would take up the viewport along with text hierarchy, making sure the titles of each section was big and visible.

What I learned

I learned how important the user is when designing a website. In a new design, I would change the color scheme to be more accessible, increasing the contrast between the colors as the white against the grey does not give enough contrast and accessibility is important to me in my designs. I would make the site more responsive to different viewports as well. Lastly, the placement and sizing of the images would be neater and consistent. Overall, the project was a great learning experience for documentation, designing, and working on a team.
